# EML Soluciones Tecnológicas - Landing Page

## Objetivo

Crear una landing page moderna, profesional y minimalista para un emprendimiento local de soporte técnico informático llamado:

EML Soluciones Tecnológicas

La página debe transmitir:

- Confianza
- Profesionalismo
- Cercanía
- Rapidez de atención
- Conocimiento técnico

NO debe parecer una empresa gamer, una startup de IA ni una página corporativa excesivamente grande.

El público objetivo son:

- Hogares
- Estudiantes
- Docentes
- Profesionales
- Pequeños comercios
- Personas con problemas tecnológicos cotidianos

Ubicación:
La Rioja Capital, Argentina.

## Filosofía del proyecto

"Esta persona sabe de tecnología y puede ayudarme"

"Esta es una gran empresa impersonal".
